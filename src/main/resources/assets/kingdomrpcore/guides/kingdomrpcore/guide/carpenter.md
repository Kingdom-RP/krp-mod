# 🪑 Плотник

Специализация пути [Ремесло](./craft.md). Доски, мебель, двери, лодки, станции, декор.

> ⚠️ Шанса провала нет. Ресурсы сгорают только при попытке скрафтить **недоступное**
> по уровню.

### Лестница доступа к крафту

| Уровень | Открывает |
|---|---|
| 0 | Доски, палки, миска, верстак, сундук, деревянные инструменты |
| 1 | Знаки, кнопки, плиты нажимные, двери, люки, лестница, удочка |
| 2 | Плиты, ступени, кровати, бочка |
| 3 | Заборы, калитки, таблички, блоки коры, бамбук-блоки, scaffolding |
| 4 | Декор: composter, рамки, картина, стойка для брони, баннеры |
| 5 | Лодки (+ с сундуком) |
| 6 | Книжные полки, улей, костры |
| 7 | Рабочие станции профессий (кафедра, ткацкий/стрелковый/картостол) |

### Эффекты

| Эффект | Формула | Пример |
|---|---|---|
| Экономия древесины | `min(50%, ур×5%)` | ур.10 = 50% |
| Партия сверх нормы | `min(40%, ур×4%)` (доски +2, палки +1) | ур.10 = 40% |
| Строительный размах | +1 блок на ур.5, +2 на ур.10 | — |

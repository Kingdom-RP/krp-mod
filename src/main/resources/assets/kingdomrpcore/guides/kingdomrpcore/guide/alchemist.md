# ⚗ Алхимик

Специализация пути [Магия](./magic.md).

### Действия для опыта

- Завершение цикла варки
- Крафт магических расходников (blaze powder, бутылки, реагенты, красители)
- Добыча натуральных цветов и глоустоуна

### Опыт за варку (по тиру эффекта)

| Тир | Зелья | XP за бутылку |
|---|---|---|
| База | awkward / mundane / thick | 2 |
| T1 | ноч. зрение, скорость, прыгучесть, мед. падение, дыхание | 5 |
| T2 | огнестойкость, лечение, яд, слабость, невидимость, замедление | 8 |
| T3 | сила, регенерация, сопротивление, вред | 12 |

> Модификатор (long/strong/splash/lingering) добавляет +2 к XP.

### Лестница доступа к варке

| Уровень | Открывает |
|---|---|
| 1 | ночное зрение, скорость, прыгучесть |
| 2 | дыхание, мед. падение, огнестойкость |
| 3 | лечение, яд |
| 4 | сила, слабость, замедление |
| 5 | регенерация; модификатор long |
| 6 | невидимость, сопротивление; модификатор strong |
| 7 | splash-зелья |
| 8 | вред (harming) |
| 9 | lingering-зелья |

### Варка — гарантированная

RNG убран. Зелье, доступное по уровню Алхимика, **варится всегда** — ни шанса
провала, ни шанса экономии реагента. Единственное ограничение — лестница доступа
выше. XP начисляется за каждую полученную бутылку.

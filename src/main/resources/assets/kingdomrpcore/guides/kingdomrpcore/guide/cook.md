# 🍖 Повар

Специализация пути [Промысел](./harvest.md).

### Действия для опыта

Готовка в печи/коптильне/на костре/в котелке FD и крафт еды.

Шкала готовки геометрическая: **XP ≈ 2 × 1.5^(тир−1)**.

### Лестница доступа к производству еды

| Уровень | Доступ |
|---|---|
| 0 | Печенье, сушёные водоросли, **хлеб** |
| 1 | Печёная картошка, жареная треска/кролик |
| 2 | Тыквенный пирог, супы/рагу, жареная курица |
| 3 | Жареный лосось/баранина, торт |
| 4 | Стейк, жареная свинина |
| 5 | Рагу из кролика |

> Активных множителей у Повара нет — специализация открывает всё более ценную пищу
> и даёт за неё опыт.

### Ограничения

- **Крафт** недоступного блюда — ресурсы сгорают.
- **Печь/коптильня** — недоступный продукт нельзя забрать.
- **Костёр** — укладка сырья отменяется.
- **Котелок FD** — блюдо нельзя извлечь.

> Найденную или купленную еду есть можно без ограничений. Farmer's Delight заведён
> в те же карты (тиры 1–10 по питательности).

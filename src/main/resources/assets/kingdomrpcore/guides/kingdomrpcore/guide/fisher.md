# 🎣 Рыбак

Специализация пути [Промысел](./harvest.md).

### Действия для опыта

Рыбалка (больше за редкое и сокровища) и сбор морской флоры.

### Опыт за улов

| Категория | XP |
|---|---|
| Хлам (палка, кожа, кость…) | 3 |
| Обычная рыба | 5 |
| Необычная | 8 |
| Редкая | 10 |
| Очень редкая | 15 |
| Сокровища / легендарная | 25 |

Морская флора (водоросли, трава, кувшинка, огурец) — 1 XP.

### Эффекты

| Уровень | Эффект |
|---|---|
| 5+ | Ускоренный клёв (+1, как ступень чары Lure) |

> Рыбалка работает через мод **Tide** — он заменяет ванильный крючок, поэтому старые
> бонусы (двойной улов, сохранение удочки и т.п.) убраны. У Tide своя система
> редкости рыбы.

### Снасти Tide (гейт по уровню Рыбака)

| Удочка | Ур. |
|---|---|
| Ванильная | 0 |
| Каменная | 1 |
| Железная | 2 |
| Золотая, кристальная | 3 |
| Алмазная | 4 |
| Echo | 5 |
| Незеритовая, призмариновая | 6 |
| Деревенская, пылающая | 7 |
| Медовая | 8 |
| Подсолнечная | 9 |
| Удочка Мидаса | 10 |

> Заброс удочки выше уровня отменяется. Крафтабельные удочки гейтятся теми же
> уровнями. Бобберы — косметика, не гейтятся.

# ⛏ Шахтёр

Специализация пути [Добыча](./mining.md).

### Действия для опыта

Добыча камня, руд и специальных блоков в обычном мире, незере и крае.

### Опыт за блоки (выборка)

| Блок | XP |
|---|---|
| Камень, булыжник, гладкий камень | 1 |
| Deepslate, булыжниковый deepslate | 2 |
| Обсидиан | 8 |
| Плачущий обсидиан | 10 |
| Угольная руда / (deepslate) | 5 / 7 |
| Железная руда / (deepslate) | 10 / 15 |
| Медная руда / (deepslate) | 10 / 15 |
| Золотая руда / (deepslate) | 12 / 18 |
| Лазурит / редстоун (deepslate) | 12 / 18 |
| Изумруд / (deepslate) | 18 / 27 |
| Алмазная руда / (deepslate) | 25 / 37 |
| Кварц / золото (незер) | 8 / 10 |
| Древние обломки | 50 |
| Скалк-семейство | 10 |
| Призмарин, морской фонарь | 5 |
| Губка (обычная и мокрая) | 5 |

> Полная таблица (песок, терракота, базальт, мох и др.) — в вики.

### Эффекты

| Уровень | Эффект |
|---|---|
| 1 | Доступ к угольной и медной рудам |
| 1+ | Скорость добычи камня и руды +5% за уровень |
| 1+ | Шанс двойного дропа с любой руды |
| 2 | Железная и золотая руды |
| 3 | Редстоун, лазурит, изумруд |
| 4 | Алмазы и обсидиан |
| 5 | Руды незера и древние обломки |

> Без уровня блок не ломается — лут и опыт не выпадают, в чат идёт требование.

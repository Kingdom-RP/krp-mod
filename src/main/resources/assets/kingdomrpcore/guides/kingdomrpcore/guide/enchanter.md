# 📖 Зачарователь

Специализация пути [Магия](./magic.md).

### Действия для опыта

Зачарование на столе, книг (ур.3), применение книг на наковальне (ур.5), извлечение
на точиле, крафт стола (ур.0).

### Опыт за зачарование (ценность чары)

| Редкость | XP за ур. I | Примеры |
|---|---|---|
| COMMON | 3 | Защита, Острота, Эффективность, Прочность |
| UNCOMMON | 5 | Заговор огня, Отбрасывание, Подводное дыхание |
| RARE | 9 | Удача, Добыча, Шипы |
| VERY_RARE | 16 | Шёлковое касание, Починка, Бесконечность |

> Острота V = 3×5 = 15 XP; Починка I = 16 XP. Точило даёт половину (не-проклятые).

### Лестница доступа (по редкости)

| Уровень | Открывает |
|---|---|
| 1 | COMMON и UNCOMMON чары |
| 3 | RARE + зачарование книг на столе |
| 5 | VERY_RARE + применение книг на наковальне |
| 7 | проклятия и чары максимального уровня (напр. Острота V) |

Число вариантов на столе: ур.0–2 → 1, ур.3–4 → 2, ур.5+ → 3.

> **Зачарование гарантированное.** Доступная по уровню чара накладывается на 100%.
> Провалов, порчи прочности и сжигания книг нет.

> **⚠ Жертва крови:** каждое успешное зачарование **на столе** забирает **2 ❤ (4 HP)**
> чистого урона (броня не спасает). Не зачаровывай на низком здоровье — может убить.

### Активные бонусы

| Где | Бонус |
|---|---|
| Стол | Усиление чары (+1 уровень): ур.3 — 9%, ур.10 — 30% |
| Стол | Экономия лазурита / сохранение опыта: ур.5 — 25%, ур.10 — 50% |
| Наковальня | Скидка на уровни: ур.5 — 25%, ур.10 — 50% |
| Наковальня | Игнор «Слишком дорого» (≥40) с ур.8 |

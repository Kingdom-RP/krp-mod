# Добро пожаловать на Kingdom RP

**Kingdom RP** — сервер, где во главе угла **разделение труда**. Нельзя в одиночку
одинаково хорошо уметь всё: чтобы ковать прочные инструменты, добывать глубокие
руды, варить сильные зелья или уверенно драться — нужно вложить время в выбранное
ремесло.

Это внутриигровой справочник. Слева — оглавление, сверху — поиск и история.
Если текст мелкий — жми **Full Width View** или **Zoom In**.

# 🧵 Мастеровой

Специализация пути [Ремесло](./craft.md). Всё, что «не металл и не дерево»: текстиль,
кожа, стекло, бетон, керамика, каменная кладка.

> ⚠️ Шанса провала нет. «Цена» низкого уровня — закрытый доступ.

### Лестница доступа к крафту

| Уровень | Открывает |
|---|---|
| 0 | Базовая стройка: текстиль/свечи, стекло, бетон, каменная кладка, глина |
| 1 | Выделка кожи, книга, вся кожаная броня |
| 2 | Черепаший шлем, лук, арбалет, крашеная терракота |
| 3 | Decorated pot, цветное стекло (витражи) |
| 4 | Узорная отделка камня (chiseled/mossy) |
| 5 | Дипслейт (полировка/кирпичи/плитка) |
| 6 | Блэкстоун, кварцевые блоки |
| 7 | Призмарин, пурпур, энд-камень-кирпичи |

> Кожаную броню **делает** Мастеровой, а **носит** Воин (с ур.1 Воина).

### Эффекты

| Эффект | Формула | Пример |
|---|---|---|
| Экономия материала | `min(50%, ур×5%)` | ур.10 = 50% |
| Партия сверх нормы | `min(40%, ур×4%)`, +2 | ур.10 = 40% |
| Закалка кожаной брони | открытие ≈⅓, открытие+3 = 100% | ботинки ур.2 ≈ 56% |

## Модовый контент

Модовые изделия гейтятся **выше** ванильных (обычно ур.5–7). **Macaw**: рубится
топором → Плотник, иначе → Мастеровой. Мосты Macaw — только гейт (ур.7), без опыта.
Handcrafted — ур.5.

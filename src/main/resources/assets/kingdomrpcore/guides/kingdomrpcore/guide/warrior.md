# ⚔ Воин

Специализация пути [Война](./war.md).

### Действия для опыта

- Убийство враждебных и нейтральных мобов в ближнем бою
- Получение урона (любого, 1 раз за тик)
- Нанесение урона мобам (кроме урона стрелами)

### Опыт за убийство моба (по тиру)

| Тир | Мобы | XP |
|---|---|---|
| S+ | Warden | 500 |
| S | Wither, Ender Dragon, Piglin Brute, Ravager | 300 |
| A | Evoker, Vex, Elder Guardian, Vindicator, Wither Skeleton, Iron Golem | 75 |
| B | Ghast, Shulker, Blaze, Hoglin/Zoglin, Enderman, Skeleton/Stray, Drowned, Cave Spider, Magma Cube, Breeze, Bogged | 15 |
| C | Zombie/Husk, Spider, Slime, Phantom, Pillager, Piglin, Zombified Piglin, Creeper | 5 |
| D | Silverfish, Endermite | 1 |

> Пассивные животные дают опыт **Промыслу** (Фермер/Рыбак), не Войне.
> См. [Промысел](./harvest.md).

### Опыт за нанесение урона

| Тир | XP за попадание |
|---|---|
| S+, S | 3 |
| A | 1.5 |
| B, C | 1 |
| D | 0.2 |
| Armor Stand | 0.05 |

**За получение урона:** 1 XP за тик (вне зависимости от величины).

### Эффекты

| Уровень | Эффект |
|---|---|
| 1 | Ношение кожаной брони (дерево/камень-оружие не ограничены) |
| 1+ | Урон в ближнем бою +2.5% за уровень |
| 1+ | Получаемый урон −2.5% за уровень |
| 2 | Кольчуга, щит, черепаший шлем |
| 3 | Железная броня и меч (лук не ограничен) |
| 4 | Золотая броня |
| 5 | Алмазное оружие и броня |
| 7 | Незеритовое оружие/броня + булава (трезубец не ограничен) |

> Без нужного уровня броня/оружие снимается автоматически с сообщением.

### Числа

| Уровень | Бонус к урону | Снижение урона |
|---|---|---|
| 1 | +2.5% | −2.5% |
| 5 | +12.5% | −12.5% |
| 10 | +25% | −25% |

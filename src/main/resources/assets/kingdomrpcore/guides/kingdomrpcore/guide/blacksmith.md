# ⚒ Кузнец

Специализация пути [Ремесло](./craft.md). Куёт инструменты, броню и утилитарку,
плавит руду, улучшает до незерита, чинит вещи.

> ⚠️ У Кузнеца **нет шанса провала крафта**. «Цена» низкого уровня — пониженная
> прочность изделия (закалка), а не сгоревшие ресурсы.

### Лестница доступа к крафту

| Уровень | Открывает |
|---|---|
| 0 | Каменные инструменты |
| 1 | Медь: громоотвод, подзорная труба, кисть, декор |
| 2 | Мелкая утилитарка: ножницы, ведро, кресало, щит, камнерез, коптильня, плавильня |
| 3 | Золотые инструменты/броня, часы, лёгкая плита; кольчуга |
| 4 | Железные инструменты/броня; двери/решётки/котёл/наковальня, компасы, поршни, вагонетки |
| 5 | Алмазные инструменты и броня |
| 7 | Незеритовый слиток, магнетит |

> Лук/арбалет крафтит **Мастеровой** (ур.2). Незеритовое снаряжение — кузнечный стол,
> **уровень 7**.

### Переплавка

| Уровень | Открывает |
|---|---|
| 1 | Медь |
| 2 | Железо, золото (+ инструменты/броня в самородки) |
| 5 | Незеритовый лом |

### Закалка (качество изделия)

Свежее изделие получает прочность по уровню: **открытие ≈ ⅓ прочности**, **открытие
+ 3 = 100%**.

| Тир | Открытие (≈33%) | Полная (100%) |
|---|---|---|
| Камень | ур.0 | ур.3 |
| Золото | ур.3 | ур.6 |
| Железо | ур.4 | ур.7 |
| Кольчуга | ур.3 | ур.6 |
| Алмаз | ур.5 | ур.8 |

> Незеритовое снаряжение прочность не понижает. Скидка на ремонт на наковальне
> растёт: ур.2 — 10%, ур.5 — 25%, ур.10 — 50%.
